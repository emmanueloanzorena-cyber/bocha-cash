# 💰 Bocha Cash

Bocha Cash es un asistente financiero personal desarrollado con Next.js y TypeScript.

## Funcionalidades

- Registro inteligente de gastos e ingresos
- Detección automática de categorías
- Detección de comercios
- Dashboard financiero
- Objetivos configurables
- Analíticas
- Chat financiero
- Copilot
- Búsqueda y filtros
- Persistencia con LocalStorage

## Tecnologías

- Next.js
- React
- TypeScript
- Tailwind CSS
- Lucide Icons
